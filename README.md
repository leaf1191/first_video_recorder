# first_video_recorder
OpenCV를 이용한 동영상 레코더

# 프로그램 기능
- 스페이스바를 눌러 녹화를 시작/종료
- +를 눌러 밝기를 증가(최대 250)
- -를 눌러 밝기를 감소(최소 -250)
